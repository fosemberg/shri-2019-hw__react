## Arcanum

### Описание

Проект создан исключительно в обучающих целях. Основная задача: повторить функциональность внутреннего проекта yandex: Arcanum.

### Запуск

#### Запуск back-end:
```npm
cd back &&
npm i &&
npm run start 
```

#### Запуск front-end:
```npm
cd front &&
npm i &&
npm run start
```
